<h1 align="center">Antinarcotics police officer registration and modification</h1>

## About the project
This project allows for the data manipulation of police officers at the service of the Antinarcotics Division. It allows for the creation of new agents, the modification of their data, and deletion from a remote database.

## Getting Started
This section will provide simple steps that will help you to run this project successfully. Please read carfully, and follow our recommendations if possible.

### Prerequisites
To run this project smoothly, you need to have this available: 
* It is possible to use any server-capable Java IDE to run this project. However, we suggest using **Eclipse**, as most of the project was developed in this environment, and it will be used from now on for the installation steps. It is also useful to have installed Spring Tools, in case you are using Eclipse.
